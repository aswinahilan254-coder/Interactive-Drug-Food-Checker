# Interactive-Drug-Food-Checker
A Python/Tkinter desktop tool for pharmacy students to check dietary interactions.

A lightweight Python desktop application built with Tkinter and Pandas. This tool provides clear dietary warnings and patient-friendly advice for over 100 common medications.

## Features
- **Search:** Case-insensitive search for drugs.
- **Severity Filters:** Filter by Major, Moderate, or Minor interactions.
- **Plain English Advice:** Converts clinical jargon into simple patient warnings.
- **Analytics:** Visual charts showing interaction trends using Matplotlib.
- **Export:** Save your search results to a CSV file.

## How to Run
1. Clone the repository.
2. Install dependencies: `pip install -r requirements.txt`
3. Run the data generator: `python create_data.py`
4. Launch the app: `python main.py`

## Disclaimer
This tool is for educational purposes (GPAT preparation) and support only. It is not a substitute for professional medical advice.
